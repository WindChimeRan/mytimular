# Timeular Macropad

![Timeular Tracker on desk](assets/IMG_0171.png)

![Menu bar UI](assets/Timeular%20Macropad.png)

Repurpose your Timeular Tracker as an 8-key macropad for macOS. Flip the tracker to a side and trigger a configurable action — launch an app, send a keyboard shortcut, run a shell command, or open a URL.
