I had a Timeular Tracker sitting on my desk, fully charged but gathering dust. I also had Claude Opus 4.6 and a head full of ideas. Even though I don't know Swift, Opus does — so I described what I wanted and let it write a native macOS menu bar app from scratch.

Now I flip my Timeular to switch macOS desktops: side 1 for my main workspace, side 2 for a side project, side 3 for another — no keyboard shortcuts to memorize, just a satisfying physical flip.
